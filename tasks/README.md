# Tasks — Housefinance

Pasta de planejamento e rastreamento de tasks para o projeto Housefinance.

## Como Usar

Cada task é um arquivo `.md` nomeado como:

```
Task-NNN-descricao-curta.md
```

Exemplo: `Task-001-viabilidade-automacao-residencial.md`

## Status Possíveis

| Status | Significado |
|--------|-------------|
| `ANÁLISE` | Investigação de viabilidade, sem código ainda |
| `PLANEJADO` | Viável, aguardando implementação |
| `EM ANDAMENTO` | Sendo implementado |
| `CONCLUÍDO` | Implementado e em produção |
| `CANCELADO` | Descartado com justificativa |

## Como Executar uma Task

Ao iniciar uma conversa com Claude para rodar uma task, basta dizer:

> "Execute a Task-NNN" ou "Rode a Task-NNN"

O Claude vai ler o arquivo, entender todo o contexto e executar.
